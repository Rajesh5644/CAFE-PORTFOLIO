# ☕ Hyderabadi Amrutulya

A modern, responsive website for **Hyderabadi Amrutulya**, designed to showcase the café's unique tea culture, menu, customer reviews, and contact information. The project focuses on delivering a clean, engaging, and mobile-friendly user experience using HTML, CSS, and JavaScript.
